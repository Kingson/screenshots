About Screenshots

Hello there! Thanks for checking out ScreenShots. It is an experiment where I explore what it takes to build a robust screenshot tool that can handles the complexity of todays websites. Please feel to try it out and make your own.

Feel free to submit bug reports on this GitHub page.

Technical deatails of this project

People are often curious about the technical details of the project such as tech stack and hosting.

Tech Stack

  • Virtual Machine (3-core VM/2Gig RAM)
  • Ubuntu Linux 18.04
  • Docker
  • Traefik for TLS and rverse proxy
  • Django
  • PostgreSQL
  • Selenium (For screenshots)
  • Tacit CSS

Hosting

Screenshots is hosted at Scaleway, a cloud services provider based in France. They offer budget-hosting solutions like the one I use here.

The server on which I run this service, the DEV1-M is a 3-core VM with 4Gigs or RAM and 20G local storage.
